    Lifelong Learning for 3D object

    Period: 2021 to 2022

    This project develops a lifelong ensemble learning framework that enables service robots to efficiently perform few-shot 3D object recognition and continuously learn new categories in open-ended environments.

    Fine-Grined 3D Object Recognition

    Period: 2022 to 2023

    This project develops a hybrid multi-modal Vision Transformer and CNN approach for fine-grained 3D object recognition in robotics, introducing new synthetic RGB-D datasets and demonstrating superior performance and practical deployment in real-world robot scenarios.

    Multiview for 3D Object Recognition

    Period: 2023 to 2024

    This project proposes the LM-MCVT network, which integrates multi-view features via the GEEF method along with convolutional encoders and local-global transformers to effectively improve 3D object recognition in robotic applications, demonstrating superior performance and robustness over state-of-the-art methods on ModelNet40 and OmniObject3D datasets.

    Hybrid Mamba-Transformer for Robot Grasp

    Period: 2024 to 2025

    This project introduces a novel Mamba-Transformer hybrid architecture that effectively integrates global and local visual features for robotic grasp detection. The proposed method achieves state-of-the-art performance across multiple benchmark datasets and demonstrates strong generalization in both simulation and real-world robot experiments.
